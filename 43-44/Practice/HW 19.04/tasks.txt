Создайте компонент списка задач, который будет использовать useState для хранения состояния выполнения каждой задачи. Позвольте пользователю отмечать задачи как выполненные и изменять их состояние при клике на элемент списка. В качестве исходного состояния используйте массив из объектов (свойства объекта: id, title, completed). Пример массива:
const tasks = [
    { id: 1, title: 'Task 1', completed: true },
    { id: 2, title: 'Task 2', completed: false },
    { id: 3, title: 'Task 3', completed: false }
  ]
