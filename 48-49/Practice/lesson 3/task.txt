Сделать так, чтобы карточки добавлялись по одной
1. Полученные данные сохранить в переменную productsData
2. Использовать глобальную переменную, которая отслеживает, сколько карточек было добавлено, и каждый раз, когда кнопка нажимается, добавлять следующую карточку.
3. Создаем переменную которая будет отвечать за индекс в productsData
currentProductIndex = 0 
Она изначально равна нулю, чтобы начать с первого элемента
4. Создаем функцию rendernextProduct
5. Условие currentProductIndex < productsData.length в этом контексте означает "пока текущий индекс продукта меньше общего количества продуктов в массиве productsData".
